# Jogo da Memória

Um jogo da memória temático, buscando enaltecer e propagar algumas das conquistas femininas mais marcantes para o desenvolvimento da humanidade nas áreas da tecnologia e da ciência. O projeto está sendo desenvolvido em **HTML, CSS e JavaScript puro**, como projeto de portfólio para praticar lógica de programação, manipulação do DOM e organização de código front-end.

## Sobre o projeto

Este projeto está sendo construído do zero como parte dos meus estudos em desenvolvimento web, unindo teoria e prática. A ideia é criar um jogo funcional, responsivo e com boa experiência de uso, aplicando conceitos como:

- Manipulação dinâmica do DOM
- Embaralhamento de elementos (algoritmo Fisher-Yates)
- Lógica de comparação e verificação de pares
- Organização de código em JavaScript puro (sem frameworks)

## Tecnologias utilizadas

- **HTML5** — estrutura da página
- **CSS3** — estilização e layout das cartas
- **JavaScript (Vanilla JS)** — lógica do jogo
